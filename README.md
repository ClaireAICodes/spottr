# Spottr

**Spottr** is a Flutter-based Progressive Web App (PWA) for gym workout planning, execution, and tracking with a focus on progressive overload and a premium glassmorphism UI.

## Documentation

- [Comprehensive Requirements](gym-workout-app-requirements.md) – Full specification including data model, UX flows, technical stack, and MVP scope.

## Status

Planning phase. This repository contains the detailed requirements document to guide future development.

## Tech Stack (Planned)

- Flutter (PWA)
- SQLite (local persistence)
- Riverpod / Bloc (state management)
- Light mode glassmorphism design
